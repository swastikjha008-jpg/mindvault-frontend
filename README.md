# 🧠 MindVault — Your Digital Second Brain

> Capture ideas. Organize thoughts. Build your knowledge system.

MindVault is a modern **Second Brain Web App** built to help users store, organize, and manage information in one clean workspace.

Whether it's **notes, ideas, tasks, reading lists, or web clips**, MindVault aims to keep everything in one place.

🚧 **Current Status:** Frontend completed • Backend integration in progress

---

## ✨ Features

### 📝 Notes
Create and organize notes in a clean UI.

### 💡 Ideas
Capture random thoughts before they disappear.

### ✅ Tasks
Track important things to do.

### 🏷️ Tags
Categorize content for better organization.

### 📚 Reading List
Save things you want to read later.

### 🌐 Web Clips
Store useful links and content.

### 🔗 Shared Feed
Planned sharing/public feed system.

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/swastikjha008-jpg/mindvault-frontend.git
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run locally

```bash
npm start
```

Open:

```bash
http://localhost:3000
```
